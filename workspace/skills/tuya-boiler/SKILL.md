# Tuya Boiler Control

To control the boiler, run the Python script:

- Turn ON: `python3 ~/.picoclaw/workspace/tuya_boiler.py on`
- Turn OFF: `python3 ~/.picoclaw/workspace/tuya_boiler.py off`
- Check status: `python3 ~/.picoclaw/workspace/tuya_boiler.py status`

Always confirm the action result to the user.
